How to build an application?

Once developement of application is done, you need to create installer of your application.

To do so, following are the steps to be followed :
  1. Copy url from GitHub [i.e. HTTPS clone URL]
  2. Go to build.phonegap.com
  3. Sign in using your GitHub login
  4. Select "+ new app"
  5. Select "Open Source" & Paste copied url to clipboard
  6. Your installer file will be create for all OS
