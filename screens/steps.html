Steps to build an application

Following are steps to be followed as it is.
  1. Create Account on GitHub
  2. “Create a new repo” in GitHub
  3. Give name to repository & create repository
  4. Click on README We recommend every repository include a README, LICENSE, and .gitignore.
  5. Change File name to “index.html”
  6. Select ”Commit new file” option.
  7. Copy URL link
  8. Start WAMP Server
  9. Go to wamp=>WWW=>Right Click=>SVN Checkout , you will find link at URL of repository or else paste it there
  10. Click on “OK”, you will find your folder name as specifies on GitHub
  11. Go to your folder eg:”mobiApp” & open it you will find [branches, trunk] folders
  12. Open Dreamweaver 5.5 =>Site=>New Site , give proper name to site & select local site folder as Wamp=>www=>”mobiApp” [your created repository]
  13. Add servers, click on “+” to add server, setting as follows:
    • Server Name : localhost
    • Connecting using :local/network
    • Server folder:c:\wamp\www\mobiApp\trunk
    • Web url : http://localhost/mobiApp/trunk
  14. After settings, click on save & testing to [select] & press “save”.

  15. Go to File=>New=>Page from sample=>MobiApp=>jQuery Mobile [Phone Gap]

  16. File saves to=>Wamp=>www=>mobiApp=>trunk=>index.html...Overwrite it
  17. You will find automatic file generates “________.js”
  18. Run "index.html" from wamp server , you will see basic layout as output.
